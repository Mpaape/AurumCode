# Technology Stack

This section documents the languages, frameworks, libraries, and tools used in the AurumCode project.

## Core Technologies

### Backend
- **Go** - Primary implementation language
- **Command execution** - Shell integration for documentation tools

### Frontend
- **Jekyll** - Static site generator
- **Just the Docs** - Documentation theme
- **GitHub Pages** - Hosting platform

### Documentation Tools
- `go doc`, `godoc` - Go documentation
- `typedoc`, `jsdoc` - JavaScript/TypeScript documentation
- `pydoc` - Python documentation
- `xmldocmd` - C# documentation
- `javadoc` - Java documentation
- `doxygen` - C/C++ documentation
- `cargo doc` - Rust documentation
- `platyPS` - PowerShell documentation

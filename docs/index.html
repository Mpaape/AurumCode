ğŸš€ Quick Start

Prerequisites

  • Go 1.21+ - Download
  • Git - Download
  • GitHub Account with repo admin access
  • API Key - OpenAI or Anthropic

Build and Run

# Clone repository
git clone https://github.com/Mpaape/AurumCode.git
cd AurumCode

# Install dependencies
go mod download

# Build
make build

# Run server
go run cmd/server/main.go

Configuration

Create .env file:

# GitHub Configuration
GITHUB_TOKEN=ghp_your_token_here
GITHUB_WEBHOOK_SECRET=your_secret_here

# LLM Provider
OPENAI_API_KEY=sk-your_key_here
# OR
ANTHROPIC_API_KEY=sk-ant-your_key_here

# Server
PORT=8080
DEBUG_LOGS=true

Run Demo

Follow the comprehensive Demo Setup Guide to see AurumCode in action!
